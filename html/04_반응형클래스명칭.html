반응형 클래스 명칭 살펴보기

단위 관련

    px(픽셀): 고정된 크기의 단위
    
    %(퍼센트):부모 요소에 비례한 상대 단위 기기 크기에 따라 자동으로 
    웹사이트 내부의 태그 요소 크기조정
    
    em/rem  : 텍스트 크기 기준 상대 단위
    
    vw/vh : 뷰포트 너비 높이의 상대 단위 vh의 경우 
    웹사이트에서 세로 크기 단위로 많이 사용

레이아웃 관련

        FlexBox : 1차원 방향으로 정렬할 때 사용하는 레이아웃
       
        Grid    : 2차원 방향으로 구성할 때 사용하는 레이아웃
      
        Container: 콘텐츠를 감싸는 부모 요소로 일반적인 고정 폭 또는 최대폭을 지정
      
        Column/Row : 그리드 시스템에서 열과 행을 의미 여러 요소를 정렬할 수 있음
            Column -> col로 사용 
                한 줄 기준으로 12개의 col이 존재
                    col-6는 절반을 차지함을 의미

            Row-> row로 사용
        
        가로 정렬 justify-content : center
        세로 정렬 align-items     : center

        마진 패딩 
        Margin -> m-숫자값 영어사용, m-1 가장 작은 단위 m-5 가장 큰 단위
        Padding-> p-5 가장 크게 테두리안에서 너비제공
        즉 border, Padding, Margin, border-radius : 1~5까지의 숫자로 표현가능
