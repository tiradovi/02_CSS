CSS(Cascading Style Sheets)

        마크업 언어(HTML)가 실제로 화면에 표시되는 방법을 기술하는 언어
        (모양, 색상, 크기, 배경이미지, 움직임,..)

        하나의 요소에 여러 CSS 코드를 겹쳐서 원하는 스타일을 만들어냄

        W3C에서 CSS를 웹 표준으로 지정

CSS 선택자(CSS Selector)

        현재 문서 내에서 특정 요소를 선택하는 문자 또는 문자열(기호)

        * 요소(Element) : 시작태그 + 속성 + 내용 + 종료태그
    *css 코드 작성법*
  • head 태그에 style 태그 작성 후 내부에 css 코드 작성
  • 원하는 요소에 style 속성을 이용해 css 코드 작성
  • 외부 .css 파일을 생성해서 연결하는 방법(현업에서 사용)

태그 선택자

        현재 문서 내에 특정 태그를 모두 찾아 선택하는 선택자

        작성법
        
            태그명{css 코드;}

id(#) 선택자

        HTML 문서 내에서
        id가 일치하는 요소를 선택하는 선택자
작성법 #id속성값{css 코드;}
  1. id 선택자 테스트1
  2. id 선택자 테스트2

class(.)선택자

        HTML 문서 내에서 class 속성 값이 일치하는 모든 요소를 선택하는 선택자
    작성법
    
        .class속성값{css 코드;}
    

    * id 보다 class 선택자 이용 권장
    (id가 CSS보다 JS에서 많이 사용되기 때문)

모든 요소 선택자(*, asterRisk)

        단독 사용 시 : 문서에 존재하는 모든 요소 선택자
        다른 선택자와 혼용 사용시 : 선택된 요소의 모든 자식/후손/형제까지 모두 적용

        CSS 적용 우선 순위 
        1. 선택자 범위가 가장 작은 선택자가 우선
        (!important > 인라인 스타일 > 아이디 > 클래스 > 태그 > 모든 요소)
        2. 태그 내에서 같은 속성값이 있을 경우 가장 마지막에 작성된 클래스 내에 존재하는 CSS 속성값이 적용
