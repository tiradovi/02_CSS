요소 영역(여백) 관련 스타일

        1. content(내용) 영역
        요소의 내용이 작성되어지는 영역
        (시작태그 종료태그 사이 작성되는 내용)

        2. padding 
        content와 border 사이의 공간

        3. border(테두리) 영역
        요소의 테두리가 지정되는 영역
        content보다 바깥 쪽에서 content를 감싸고 있음

        4. margin(여백, 간격)
        

        
        box-sizing
        width/height 크기 속성의 범위를 지정하는 css 속성

            1) content-box 
            width/height 지정 시 content 영역의 크기만 조정
            (기본값)

            2) border-box
            width/height 지정시
            content + padding + border 의 크기를 합산한 결과가 
            작성된 width/height와 같아지도록 content 영역의 크기를 자동으로 조정

padding, margin , border 의 방향성

      000==padding, margin , border 를 의미
      000-top : 크기 -> 위쪽 방향 크기 지정
      000-bottom : 크기 -> 아래쪽 방향 크기 지정
      000-left : 크기 -> 왼쪽 방향 크기 지정
      000-right : 크기 -> 오른쪽 방향 크기 지정

      border의 속성 값은 [크기, 모양, 색상] 형식으로 작성
content
      padding | margin | border-width | border-radius : 속성 값(크기)
      속성 값 1개 : 상하좌우 한번에 모두 동일한 값 작성
      속성 값 2개 : 상하 좌우 두 가지로 나누어서 작성(button padding 에서 많이 사용)
      속성 값 3개 : 상 좌우 하 세 가지 (상 -> 좌우 -> 하)
      속성 값 4개 : 상 우 하 좌 네 가지 (상 -> 우 -> 하 -> 좌)
