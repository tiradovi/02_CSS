Grid - 레이아웃 스타일

        사전적 의미 : 격자, 바둑판 무늬
        기술적 의미 : 2차원(행과 열)로 웹페이지를 격자형태로 나누어 배치하는 레이아웃

        [Grid 특징]
        행(row)과 열(column)을 동시에 제어
        복잡한 레이아웃을 간단하게 표현
        반응형 디자인에 최적화
        FlexBox보다 2차원 레이아웃에 적합

Grid Container

        display : grid 가 적용된 부모 요소

Grid Item

        부모 요소 내부에 grid가 적용된 자식 요소

기본 구조

.container { display: grid; /*그리드 컨테이너 생성*/
grid-template-columns: 1fr 1fr; /* 열 정의 */
grid-template-rows : 1fr 1fr; /* 행 정의 */
gap : 10px; /* 간 격 */
}

기본 2열 그리드

2개의 열을 균등하게 설정 grid-template-columns: 1fr 1fr;

3열 그리드 (다른 비율)

grid-template-columns: 1fr 2fr 1fr

고정 크기와 가변 크기 조합

grid-template-columns: 200px 1fr 150px; /*고정-가변-고정*/ 고정 : 가로 너비에 관계없이 크기 고정 세로 너비에 관계없이 크기 고정 가변 : 기기 화면 너비에 따라 너비가 변화
